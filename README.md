# LibAR — 서가 스캔 데모 (2026 도서관 데이터 활용 공모전)

스마트폰/PC 브라우저에서 도서관 서가를 스캔해 청구기호를 인식하고 오배열을 찾아내는
**온디바이스 AI** 데모입니다. 검출→인식→장서 대조→오배열 판정까지 전 과정이 브라우저 안에서
이루어지며 사진은 서버로 전송되지 않습니다.

**데모:** https://ainsof.dev/libar-demo/

## 기능
- **서가 점검하기**: 카메라를 비추면 실시간 라벨 검출 → 일시정지하면 멈춘 화면에서 청구기호를
  읽어 🟢확인 / 🔴잘못 꽂힘 / ⚪다시 확인으로 판정 (약 20~30초, 진행률 표시)
- **도서 찾기**: 장서 1,860권 검색 → 층별 실배치도 길안내 → 서가에서 책 위치 하이라이트
- **샘플 서가**: 촬영해 둔 대림도서관 서가 2장으로 전체 흐름 바로 체험
- **학습용 조각 제공(선택)**: 동의하면 라벨 조각만 팀 드라이브로 전송해 모델 개선에 사용

## 온디바이스 모델
| 단계 | 모델 | 크기 |
| :-- | :-- | :-- |
| 라벨 검출 | YOLO26n (자가 학습 v3) | 10MB |
| 텍스트 줄 검출 | PP-OCRv5 mobile det | 4.5MB |
| 청구기호 인식 | 한국어 rec, 저해상 라벨 파인튜닝 v4 (fp16) | 48MB |

ONNX Runtime Web — WebGPU 우선, 미지원 기기는 WASM 자동 폴백 (판독 결과 동일).
첫 실행 시 모델을 내려받아 이후엔 캐시로 동작합니다.

실증: 영등포구립 대림도서관 · 장서 정보는 공개 서지 데이터입니다.
