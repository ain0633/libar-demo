# LibAR — 서가 스캔 데모 (2026 도서관 데이터 활용 공모전)

스마트폰/PC 브라우저에서 도서관 서가를 스캔해 청구기호를 인식하고 오배열을 찾아내는
온디바이스 AI 데모입니다. 모든 처리는 브라우저 안에서 이루어지며 사진은 전송되지 않습니다.

**데모:** https://ain0633.github.io/libar-demo/

- 서가 점검하기: 카메라를 비추면 실시간으로 책 라벨을 검출 (YOLO26n ONNX, WebGPU/WASM)
- 판정 시연: 실측 파이프라인 결과(청구기호 인식·오배열 판정) 오버레이
- 도서 찾기: 장서 1,860권 검색 → 층별 지도 길안내 → 서가에서 책 위치 하이라이트

실증: 영등포구립 대림도서관 · 장서 정보는 공개 서지 데이터입니다.
